Arquivos de áudio

Aqui aprendemos a implementar arquivos de áudio como músicas em MP3 ao nosso site

Sempre quando implementamos um arquivo de áudio como éo caso ou de vídeo devemos sempre colocar em mais de um formato para que o site fique com compatibilidade como todos os navegadores. Os formatos mais comuns para áudio são MP3,WAV, e OGG

Quando fomos cadastrar outros formatos não ultilizamos a TAG SRC . Nós ultilizamos a TAG AUDIO normal e em seguida abrimos dentro dela a TAG SOURCE com a especificação SRC , vai acrescentar o arquivo em mp3 ou em outro formato na nomeclatura src manuamente e na TAG TYPE você deve colocar o nome audio/é o formato desejado

EX:audio/mpeg para MP3 audio/ogg para arquivos OGG audio/wav para arquivos WAV

Sempre devemos por nessa Ordem pois o site vai tentar abrir primeiro com MP3 depois com OGG depois com WAV vai depender da compatibilidade de cada navegador, como o MP3(mpeg) é o mais compativel com qualquer navegador deve sempre abrir primeiro.

Para ficar sempre repetindo o áudio devemos colocar a TAG LOOP. É para ativar o play devemos incluir a TAG CONTROLS É para tocar automaticamente assim que entrarmos no site devemos colocar a TAG AUTOPLAY (a qual nao funciona com qualquer navegador por isso devemos sempre colocar os controles para facilitar a ultilização)

Devemos sempre abrir um paragráfo ao fim de cadastrar todos os formatos possiveis dentro da TAG AUDIO para fim de informação.

EX: Este seu navegador infelizmente não consegue reproduzir o áudio do site.
